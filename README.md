# Switch Cheats for Atmosphere and Eden

## General infos
This repository contains cheat files for Nintendo Switch dumped from [titledb](https://github.com/blawar/titledb) repository from [blawar](https://github.com/blawar), usable onto [Atmosphere](https://github.com/Atmosphere-NX/Atmosphere) and Eden Emulator.<br>

To use these cheats into Atmosphere, copy your needed TitleIDs from <code>atmosphere/titles</code>, and to use them into Eden emulator, copy your needed TitleIDS from <code>files/load</code>.<br>

The Python scripts used for dumping cheats are included into this repo, these scripts use the <code>cheats.json</code> file and localized game names JSON files retrieven from [titledb](https://github.com/blawar/titledb) repository (e.g: <code>US.en.json</code>), that must be saved into the <code>label_files</code> folder. <br> For each TitleId, the localized game name is saved as an <code>.txt</code> file into the TitleId folder.<br><br>

Localized game names JSON files are **not** included into this repository in order to save space.
