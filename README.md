# Switch Cheats for Atmosphere and Eden

This repository contains cheat files dumped from [titledb](https://github.com/blawar/titledb) repository from [blawar](https://github.com/blawar), usable onto [Atmosphere](https://github.com/Atmosphere-NX/Atmosphere) and Eden Emulator.

To use these cheats into Atmosphere, copy your needed TitleIDs from <code>atmosphere/titles</code>, and to use them into Eden emulator, copy your needed TitleIDS from <code>files/load</code>.

The Python scripts used for dumping cheats are included into this repo, these scripts use the <code>cheats.json</code> file and localized game names retrieven from [titledb](https://github.com/blawar/titledb) repository. Localized game names are not included into this repository in order to save space.
